# is-number

Returns true if a value is a number and not NaN.

```javascript
module.exports = x => typeof x === 'number' && !Number.isNaN(x)
```

## Installation

Requires [Node.js](https://nodejs.org/) 4.0.0 or above.

```bash
npm i @lamansky/is-number
```
